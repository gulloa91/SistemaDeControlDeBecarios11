Insertar

Ver [Ver sección 2.4.3]. Para agregar una asignación al sistema, se indican los siguientes datos:

·         Total de horas

·         Encargado

·         Becarios sin asignación

·         Unidad académica

·         Información de ubicación

·         Comentario de la Dirección

Figura 49. Edición de una nueva asignación

Modificar

La acción de modificar una asignación se corresponde con uno de los casos especiales dentro del sistema [Ver sección 2.4.4]. Existen dos modos para ello:

·         Realizar comentario: habilita el campo “Comentario de la Dirección”, para poder emitir un comentario sobre la asignación actual.

·         Cambiar asignación: de manera similar a cuando se realiza una inserción, permite cambiar todos los campos de la asignación (excepto los comentarios realizados por el becario y el encargado). Esta operación sólo se puede realizar cuando alguna de las partes ha rechazado la asignación; de lo contrario, el sistema le informará de la invalidez de tal cambio.

Eliminar

En caso de querer deshacer una asignación, existe la opción de eliminarla del sistema [Ver sección 2.4.5].
