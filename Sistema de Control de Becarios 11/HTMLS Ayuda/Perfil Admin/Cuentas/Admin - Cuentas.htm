Vista principal

En la pantalla principal se le va a presentar una tabla con todas las cuentas que están en el sistema hasta el momento [Ver sección 2.4.1].

Puede seleccionar una cuenta existente para ver sus datos [Ver sección 2.4.2]:

Insertar

Cuando se agrega un becario o un encargado al sistema, se le asignará una cuenta automáticamente. Esta funcionalidad existe cuando sea necesario asignar de vuelta una cuenta previamente eliminada, o agregar administradores. Solo un usuario con perfil de Administrador de la aplicación puede crear cuentas.

Ingrese al modo de inserción de cuenta [Ver sección 2.4.3]. Para agregar una cuenta al sistema, se indican los siguientes datos:

·         Nombre de usuario

·         Contraseña

·         Confirmación de contraseña

·         Tipo de cuenta:

o   Administrador

o   Becario

o   Encargado

Si indica que la cuenta es de tipo Becario o Encargado, aparecerá un campo adicional, que le permitirá escoger de entre una lista a un becario o encargado (dependiendo de su escogencia) que no tenga una cuenta asociada.

Modificar

El botón “Modificar” habilita la edición de todos los campos de una cuenta [Ver sección 2.4.4].

 

Eliminar

En caso de que no se desee mantener en existencia a una cuenta, existe la opción de borrarla del sistema [Ver sección 2.4.5]. No puede borrarse la cuenta activa. Para poder borrar una cuenta de becario o de encargado, el mismo no puede estar en medio de una asignación, o con asignaciones pendientes.
